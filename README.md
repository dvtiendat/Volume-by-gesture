# Adjust volume by gesture

*Introduction*

Here this the python code for adjusting volume by using hand gesture. By stretch 2 fingers (point and thumb respectively), you can volume up and volume down the system's volume.

- cv2
- mediapipe, pyautogui
- python 3.9.10
